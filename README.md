# 🚀 Prompt Engineering with Ollama (Python + Local LLM)

A collection of hands-on **Prompt Engineering** projects built using **Python** and **Ollama** with the **llama3.2** local language model.

This repository was created as part of my **Generative AI training**, where I explored how different prompting techniques influence the behavior and quality of responses generated by Large Language Models (LLMs).

Unlike cloud-based AI applications, every project in this repository runs **locally** using Ollama, making it lightweight, private, and free from external API dependencies.

---

# 📌 Objectives

* Learn practical Prompt Engineering techniques
* Understand how prompt design affects LLM responses
* Work with a local LLM using Ollama
* Build real-world terminal-based AI assistants
* Explore structured system prompt design

# 📖 Projects Included

## 1️⃣ Zero-Shot Prompting

### Meeting Notes Summarizer

Converts unstructured meeting notes into an organized summary.

### Features

* Topic extraction
* Key discussion points
* Decisions made
* Action items
* Deadlines
* Additional notes

The prompt follows strict anti-hallucination instructions to avoid generating information that is not present in the input.

---

## 2️⃣ One-Shot Prompting

### LinkedIn Post Formatter

Uses one example before generating the final output.

The assistant converts a project description into a professional LinkedIn post while maintaining an engaging writing style.

---

## 3️⃣ Few-Shot Prompting

### Bug Severity Classifier

Uses multiple labeled examples before classification.

The assistant predicts whether a software bug is:

* Critical
* High
* Medium
* Low

It also provides a short explanation for the assigned severity.

---

## 4️⃣ Chain-of-Thought Prompting

### Loan EMI Explanation Assistant

Calculates:

* Monthly EMI
* Total Payment
* Total Interest

Instead of returning only the answer, the assistant explains each calculation step in a structured manner.

---

## 5️⃣ ReAct Prompting

### Career Path Recommendation Assistant

Implements the **ReAct (Reason + Act)** prompting pattern.

The assistant evaluates different career options through:

* Thought
* Action
* Observation

before recommending the most suitable career paths.

# ▶️ Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

## 2. Navigate to a Project

```bash
cd ZeroShot
```

or

```bash
cd OneShot
```

---

## 3. Create a Virtual Environment

Windows

```bash
python -m venv .venv
```

Activate

```bash
.venv\Scripts\activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5. Install Ollama

Download and install Ollama from the official website.

After installation, pull the required model:

```bash
ollama pull llama3.2
```

---

## 6. Configure Environment Variables

Create a `.env` file.

Example:

```env
MODEL_NAME=llama3.2
TEMPERATURE=0.3
```

---

## 7. Run the Project

```bash
python main.py
```
